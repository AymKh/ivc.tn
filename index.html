Free Open Source Game

What is it ?

IVC is a free open source 3d game made by 2 students for a uni project, using 3dsMax, Unity 3D & Pro Builder. This game is nowhere near finished, as it is a project on hold. However, you are free to download the source files, tweak it, rebuild it and even distribute it. or even better, you can reuse the assets to make a completey different game of your own.

ISLAIB

( Institut supérieur des langues appliquées et d’informatique de Béja ) University used as a refernce for the main map models. Maybe you can model part of your own Uni. you can learn more about this University here
